Top Window  

 

  • To build a window that sits on top of all other windows, a Toplevel widget is utilized. 
  • The Toplevel widget delivers additional information to the user when our software is dealing with multiple applications. 
  • The Window Manager organizes and manages these windows directly, so they don't need to be paired with a parent window every time.
  • Syntax -

toplevel = Toplevel(root, bg, fg, bd, height, width, font, ..)
